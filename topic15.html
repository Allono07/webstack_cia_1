HTML Media

HTML5 introduced several new elements that make it easier to embed and handle media content like audio and video on web pages. This guide covers the key HTML5 media elements and attributes, as well as examples of their usage.

Audio

The <audio> element is used to embed audio content on a web page. You can include multiple source files in different formats to ensure compatibility across different browsers.

<audio controls>
  <source src="audio-file.mp3" type="audio/mpeg">
  <source src="audio-file.ogg" type="audio/ogg">
  Your browser does not support the audio element.
</audio>

Video

The <video> element is used to embed video content. Similar to the <audio> element, you can include multiple source files to ensure compatibility.

<video controls width="600">
  <source src="video-file.mp4" type="video/mp4">
  <source src="video-file.ogg" type="video/ogg">
  Your browser does not support the video element.
</video>

Attributes

  • controls: Adds playback controls like play, pause, and volume.
  • autoplay: The media will start playing as soon as it is ready.
  • loop: The media will loop back to the start after finishing.
  • muted: The media will start muted.
  • preload: Specifies if and how the media should be loaded when the page loads. Values can be auto, metadata, or none.

Example: Audio Player

<audio controls>
  <source src="song.mp3" type="audio/mpeg">
  <source src="song.ogg" type="audio/ogg">
  Your browser does not support the audio element.
</audio>

Example: Video Player

<video controls width="600">
  <source src="movie.mp4" type="video/mp4">
  <source src="movie.ogg" type="video/ogg">
  Your browser does not support the video element.
</video>

Captions and Subtitles

To add captions or subtitles to your video, use the <track> element. It allows you to specify different kinds of timed text tracks, including subtitles, captions, and descriptions.

<video controls width="600">
  <source src="movie.mp4" type="video/mp4">
  <track src="subtitles_en.vtt" kind="subtitles" srclang="en" label="English">
  Your browser does not support the video element.
</video>

Embedding Media from External Sources

HTML5 also allows for easy embedding of media from external sources, such as YouTube or Vimeo, using the <iframe> element.

<iframe width="560" height="315" src="https://www.youtube.com/embed/dQw4w9WgXcQ" 
  frameborder="0" allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" 
  allowfullscreen></iframe>

Conclusion

HTML5 media elements like <audio> and <video> provide a standardized way to embed and control media content on web pages. By leveraging these elements and their attributes, developers can create rich, interactive multimedia experiences that work across all modern browsers.
