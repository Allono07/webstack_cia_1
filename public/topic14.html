Canvas

The HTML5 <canvas> element is used to draw graphics on a web page via JavaScript. It provides a surface for rendering graphics on the fly, including shapes, text, images, and animations. This guide explores the basics of the <canvas> element and its uses in web development.

The <canvas> Element

The <canvas> element is a container for graphics. You must use JavaScript to actually draw the graphics.

<canvas id="myCanvas" width="400" height="400">
  Your browser does not support the canvas element.
</canvas>

Drawing on Canvas

To draw on the <canvas> element, you use the getContext() method to get the rendering context and its drawing functions.

<script>
  var canvas = document.getElementById("myCanvas");
  var ctx = canvas.getContext("2d");
  ctx.fillStyle = "#FF0000";
  ctx.fillRect(0, 0, 150, 75);
</script>

This code selects the <canvas> element with the id "myCanvas" and sets the fill color to red. It then draws a rectangle starting at coordinates (0, 0) with a width of 150 pixels and a height of 75 pixels.

Basic Drawing Functions

  • Rectangles: fillRect(x, y, width, height) draws a filled rectangle.
  • Paths: Use beginPath(), moveTo(x, y), lineTo(x, y), and stroke() to draw lines.
  • Arcs: arc(x, y, radius, startAngle, endAngle) draws an arc/curve.
  • Text: fillText(text, x, y) and strokeText(text, x, y) draw filled and outlined text, respectively.

Example: Drawing a Circle

<script>
  var canvas = document.getElementById("myCanvas");
  var ctx = canvas.getContext("2d");
  ctx.beginPath();
  ctx.arc(100, 75, 50, 0, 2 * Math.PI);
  ctx.stroke();
</script>

This code draws a circle with a center at (100, 75), a radius of 50 pixels, and a full 360-degree circumference.

Animations

The <canvas> element can also be used to create animations by continuously updating the canvas content using JavaScript's requestAnimationFrame() method.

<script>
  var canvas = document.getElementById("myCanvas");
  var ctx = canvas.getContext("2d");
  var x = 0;
  var y = 0;
  function draw() {
    ctx.clearRect(0, 0, canvas.width, canvas.height);
    ctx.fillRect(x, y, 50, 50);
    x++;
    y++;
    requestAnimationFrame(draw);
  }
  draw();
</script>

This code moves a 50x50 pixel square diagonally across the canvas.

Advanced Techniques

  • Images: Use the drawImage() method to draw images onto the canvas.
  • Transformations: Use methods like rotate(), scale(), and translate() to transform the canvas context.
  • Compositing: Use global composite operations to create complex visual effects.
  • Pixel Manipulation: Access and manipulate individual pixels using the getImageData() and putImageData() methods.
